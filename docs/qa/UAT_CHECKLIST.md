# UAT Checklist

## File lifecycle
- [ ] Create new project.
- [ ] Save as `.pmgov`.
- [ ] Reopen saved file.
- [ ] Open invalid file and confirm error.

## Workstreams
- [ ] Create workstream.
- [ ] Create stage.
- [ ] Create milestone.
- [ ] Edit milestone forecast date.
- [ ] Confirm variance appears.

## Notebook
- [ ] Create meeting note.
- [ ] Edit note.
- [ ] Add tag.
- [ ] Create action from note.
- [ ] Create decision from note.

## Governance
- [ ] Create action manually.
- [ ] Mark action complete.
- [ ] Create decision manually.
- [ ] Link decision to note.

## Dashboard
- [ ] Confirm delayed milestone appears in attention list.
- [ ] Confirm overdue action appears.
- [ ] Confirm recent decision appears.

## Reporting
- [ ] Generate executive report.
- [ ] Copy markdown.
- [ ] Print HTML view.
