# Task 03 — App Shell and Navigation

## Goal
Build the main layout and navigation.

## Requirements
- Start screen when no file is open.
- Sidebar when project is open.
- Top bar with project name, Open, Save, Save As, unsaved indicator.
- Pages: Dashboard, Workstreams, Timeline, Notebook, Governance, Reports, Settings.

## Acceptance criteria
- Navigation works.
- Unsaved indicator appears after edits.
- App remains usable with empty project.
