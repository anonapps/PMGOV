# Task 08 — Reporting

## Goal
Generate executive status report.

## Requirements
- Build report from current project state.
- Sections: health, summary, workstream status, milestones, decisions, actions.
- Provide Copy Markdown.
- Provide printable HTML view.

## Acceptance criteria
- Report can be generated without errors for sample project.
- Report handles empty sections gracefully.
- Copied markdown is clean and executive-readable.
