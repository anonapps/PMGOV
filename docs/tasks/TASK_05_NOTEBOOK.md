# Task 05 — Notebook

## Goal
Implement note capture.

## Requirements
- Create/edit/delete notes.
- Note fields: title, type, content, tags, createdAt, updatedAt.
- Basic rich text acceptable; markdown textarea acceptable if rich editor delays MVP.
- Link note to created action/decision.

## Acceptance criteria
- Notes persist in `.pmgov`.
- Notes can be searched by title or tag.
- Note can be used as source for action/decision creation.
