# Task 01 — Repository Setup

## Goal
Create the base Next.js application.

## Requirements
- Next.js App Router.
- TypeScript strict mode.
- Tailwind CSS.
- ESLint.
- No backend persistence.
- Add `/docs`, `/schemas`, `/examples`, `/src/types`.

## Acceptance criteria
- App runs locally.
- TypeScript compiles.
- Tailwind works.
- README explains local-first constraint.
